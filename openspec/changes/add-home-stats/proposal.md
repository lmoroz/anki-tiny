# Proposal: Add Aggregated Training Statistics to Home Page

## Problem

Текущая главная страница (`HomePage.vue`) не предоставляет пользователю информации об общем прогрессе обучения:

1. **Отсутствие обзора прогресса**: Пользователь не видит, сколько карточек изучено/повторено за день по всем курсам.
2. **Нет информации о лимитах**: Неясно, сколько новых карточек доступно сегодня по всем курсам.
3. **Нет мотивации**: Отсутствуют метрики для отслеживания ежедневной активности (количество тренировок).
4. **Неэффективное использование пространства**: Главная страница показывает только список курсов, оставляя много
   свободного места.

### Current State

- Главная страница содержит только список курсов в сетке (`CourseList` компонент).
- Статистика доступна только на уровне отдельных курсов (через `CourseCard`).
- Глобальная статистика за день доступна через API (`GET /api/training/stats`), но не отображается в UI.

## Why

Глобальная статистика на главной странице критична для мотивации пользователей и эффективного планирования учебного процесса:

1. **Отслеживание прогресса**: Пользователям необходимо видеть общий прогресс обучения в одном месте, без необходимости
   переключаться между отдельными курсами.
2. **Понимание лимитов**: Чёткое отображение дневных лимитов помогает пользователям планировать время на обучение.
3. **Мотивация**: Визуализация выполненной работы (количество тренировок) стимулирует продолжать обучение ежедневно.
4. **Эффективность интерфейса**: Использование двухколоночного layout на desktop экранах оптимизирует пространство и
   позволяет одновременно видеть список курсов и ключевые метрики.
5. **Подготовка к расширению**: Выделенная область для статистики создаёт основу для будущих улучшений (графики,
   детализация по дням).

## Solution

Разделить главную страницу на две колонки:

1. **Левая колонка (50%)**: Список курсов (существующий `CourseList`)
2. **Правая колонка (50%)**: Агрегированная статистика тренировок

### Статистика будет включать

- **Новые карточки**: Общее количество новых карточек по всем курсам
- **Изучено/Повторено сегодня**: Сумма новых карточек и повторений за день
- **Осталось на сегодня**: Сколько карточек ещё осталось для изучения/повторения (с учётом лимитов)
- **Дневной лимит новых карточек**: Глобальный лимит новых карточек в день
- **Количество тренировок сегодня**: Общее количество завершённых повторений (newCardsStudied + reviewsCompleted)

В будущем (вне scope этого proposal):

- График статистики тренировок по дням

## Benefits

- ✅ **Мотивация**: Пользователь видит свой ежедневный прогресс в одном месте
- ✅ **Прозрачность**: Чёткое понимание лимитов и доступных карточек
- ✅ **Эффективное использование пространства**: Desktop layout оптимально использует ширину экрана
- ✅ **Готовая основа**: Подготовка места для будущего графика статистики

## Scope

### In Scope

- Добавление нового API endpoint: `GET /api/stats/global` (или использование существующего `/api/training/stats`)
- Создание компонента `GlobalStats.vue` для отображения статистики
- Изменение layout `HomePage.vue` на двухколоночный (для desktop ≥1024px)
- Адаптивный дизайн для mobile (single column, статистика выше списка курсов)
- Расчёт агрегированных метрик:
    - Общее количество новых карточек (по всем курсам)
    - Изучено/повторено сегодня (newCardsStudied + reviewsCompleted)
    - Осталось на сегодня (с учётом глобальных и курсовых лимитов)
    - Глобальный лимит новых карточек
    - Общее количество тренировок за день
- Создание нового spec для главной страницы: `home-stats` (или расширение `course-ui`)

### Out of Scope

- График статистики тренировок по дням (отдельный proposal)
- Детализация статистики по курсам на главной странице (доступна на `CoursePage`)
- Фильтрация и сортировка курсов
- Настройки отображения статистики

## Migration Notes

> [!NOTE]
> Этот proposal не изменяет существующий функционал, только добавляет новую панель статистики.
